Panduan Instalasi

Instalasi Code-Server (Recommended)

Metode ini mendownload binary yang sudah dicompile, menghemat waktu dan resource CPU.

1. Clone Repository & Masuk Direktori

git clone https://github.com/biezz-2/vscode-server-deploy.git cd vscode-server-deploy

2. Jalankan Installer

chmod +x install-code-server-microsoft-marketplace.sh ./install-code-server-microsoft-marketplace.sh

Akses server di: http://IP_ADDRESS:8680

Instalasi VS Code Server Original

Metode ini melakukan build source code dari nol. Pastikan server memiliki RAM min 4GB.

1. Clone Repository & Masuk Direktori

git clone https://github.com/biezz-2/vscode-server-deploy.git cd vscode-server-deploy

2. Jalankan Installer (Proses Lama)

chmod +x install-vscode-server-original.sh ./install-vscode-server-original.sh

Proses ini memakan waktu 30-60 menit tergantung CPU.

Manajemen Service

Gunakan perintah systemd berikut untuk mengelola server.

Perintah Dasar
Start Service systemctl start code-server@$USER
Restart systemctl restart code-server@$USER
Check Logs journalctl -u code-server@$USER -f

Keamanan (Wajib)

Jangan biarkan server terbuka tanpa password.

1. Set Password

nano ~/.config/code-server/config.yaml

Ubah auth: none menjadi auth: password dan isi password Anda.

2. HTTPS / SSL

Dibutuhkan agar fitur copy-paste browser berfungsi.

openssl req -x509 -nodes -days 365 ...

Troubleshooting & FAQ

Cek dengan netstat -tulpn | grep 8680. Jika ada yang pakai, edit file config ~/.config/code-server/config.yaml dan ubah bind-addr ke port lain (misal 9000). Restart service setelahnya.
Gunakan perintah journalctl -u code-server@$USER -n 50 untuk melihat log error terakhir. Biasanya masalah salah ketik di config.yaml atau port bentrok.
