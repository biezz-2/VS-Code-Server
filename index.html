Instalasi

Instalasi Cepat (Code-Server)

Salin dan jalankan perintah ini di terminal server Anda:

bash -c "$(curl -fsSL https://raw.githubusercontent.com/biezz-2/VS-Code-Server/refs/heads/main/unofficial-vscode-server.sh)"

Akses browser: http://YOUR_IP:8680

Instalasi Original (Build Source)

Proses ini memakan waktu lama. Pastikan RAM > 4GB.

bash -c "$(curl -fsSL https://raw.githubusercontent.com/biezz-2/VS-Code-Server/refs/heads/main/official-vscode-server.sh)"

Akses browser: http://YOUR_IP:8680

Manajemen Service

Gunakan perintah systemd berikut untuk mengelola server.

Perintah Dasar
Start Service systemctl start code-server@$USER
Restart systemctl restart code-server@$USER
Check Logs journalctl -u code-server@$USER -f

Keamanan (Wajib)

Jangan biarkan server terbuka tanpa password. **Direkomendasikan.**

1. Set Password

nano ~/.config/code-server/config.yaml

Ubah `auth: none` menjadi `auth: password` dan isi password Anda. Lalu restart service.

2. HTTPS / SSL

Dibutuhkan agar fitur copy-paste browser berfungsi dan untuk keamanan data.

openssl req -x509 -nodes -days 365 ...
